# Haskell: Data Type

## Data Type

- **Int**
  - Integer in 64-Bits range.
- **Integer**
  - Any Integer.
- **Double**
  - Decimal Number.
- **Bool**
  - Truth Value (True, False)
- **Char**
  - Single Character that's denoted by the single quote ('')
- **String**
  - Set of the character that's denoted by the double quote ("")
